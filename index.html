Projetos

Criado por RToramaru 2022.

registro terapêutico

O Registro Terapêutico é um aplicativo que tem como objetivo registrar, consultar e gerar relatórios sobre os atendimentos realizados por profissionais de saúde. O aplicativo foi desenvolvido em Flutter e utiliza uma API externa para armazenar os dados.

Calculadora catalisador

O Calculadora Catalisador é um aplicativo que tem como objetivo calcular o preço da sucata de catalisador com base na proporção de paladio, platina e ródio contida na peça. O aplicativo foi desenvolvido em Flutter e utiliza uma API externa para armazenar os dados.

Cérbero

O Cérbero é um sistema de controle de acesso de veículos desenvolvido em Python utilizando redes neurais convolucionais para o reconhecimento das placas veiculares. O sistema foi desenvolvido para utilização do IFNMG - Campus Salinas como forma de controle de acesso de veículos.

Mensageiro

Mensageiro é um aplicativo de mensagens instantâneas desenvolvido em Flutter e utilizando do FireBase como banco de dados. Nele é possível enviar mensagens de texto, imagens e vídeos para os contatos cadastrados.

Chat em grupo

Chat em grupo é um aplicativo de mensagens instantâneas desenvolvido em Flutter e utilizando do FireBase como banco de dados. Nele o administrador pode criar grupos e todos os usuários cadastrados podem enviar mensagens de texto, imagens e vídeos para os grupos.

Clima Tempo

Clima tempo é um aplicativo de previsão do tempo desenvolvido em Flutter e consumindo Weather API. Nele é possível ver a previsão do tempo para o local atual obtido através da localização, ou então é possível obter a previsão do tempo para qualquer cidade do mundo.

Delivery App

Delivery App é um aplicativo de delivery de comida desenvolvido em Flutter e utilizando do FireBase como banco de dados. Nele é possível comprar produtos ficticios, favoritar produtos, realizar pedidos, ver histórico de pedidos e muito mais.

RToramaru E-commerce

RToramaru E-commerce é uma aplicação que simula um e-commerce, desenvolvido em PHP utilizando do framework Laravel e utilizando do banco de dados MySQL. Na aplicação é possível realizar cadastro de usuários, realizar login, adicionar produtos ao carrinho, realizar pedidos, ver histórico de pedidos, validar o cartão de crédito e muito mais.

RToramaru Eventos

RToramaru Eventos é uma aplicação que simula um site de eventos, desenvolvido em PHP utilizando do framework Laravel e utilizando do banco de dados MySQL. Na aplicação é possível realizar cadastro de usuários, realizar login, criar evento, participar de eventos, calcelar participação, cancelar evento, ver histórico de eventos e muito mais.

Contador Pomodoro

Contador Pomodoro é um aplicativo que simula o método pomodoro, desenvolvido em Flutter. O aplicativo é capaz de realizar o controle de tempo, realizar o controle de tarefas, realizar o controle de pausas e muito mais.

Preço do catalisador

Preço do catalisador é um aplicativo desenvolvido em Flutter que calcula o preço dos catalisadores, com base no código do catalisador e com o preço dos metais assim como o preço das principais moedas.

SICOIN

SICOIN ou Sistema de Controle de Insumos é um sistema desenvolvido em PHP para utilização do setor financeiro do IFNMG - Campus Salinas para controle interno da central de vendas, no qual é possível realizar o controle de estoque, realizar o controle de vendas, realizar o controle de compras, realizar o controle de fornecedores, realizar o controle de clientes, realizar o controle de usuários, realizar o controle de categorias de produtos, realizar o controle de produtos, realizar o controle de caixa, realizar o controle de contas a pagar e muito mais.
